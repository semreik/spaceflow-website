# SpaceFlow Website

Modern, responsive website for SpaceFlow's intelligent procurement orchestration platform.

## Features

- **Next.js 14.2.5** with TypeScript
- **Framer Motion** animations with reduced motion support
- **Tailwind CSS** for responsive styling
- **Mobile-optimized** layouts and interactions
- **Clean workflow** orchestration demo with slide-replace animations

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Key Components

- `Hero` - Landing page hero section with tour modal
- `CleanWorkflow` - Interactive workflow demo with procurement scenarios
- `StrategicPartners` - Animated partners ticker
- `AlternatingFeatures` - Feature showcase sections
- `Integrations` - Integration partners display
- `Security` - Security features section

## Deployment

The site is optimized for deployment on Vercel, Netlify, or any static hosting platform.
